## 📌 Project Overview
This project solves the **Interval Scheduling Problem** using two algorithmic approaches applied to a real-world cloud task scheduling dataset.

## ⚙️ Algorithms Implemented
1. **Greedy Algorithm (Earliest Finish Time)** – maximizes number of non-overlapping tasks
2. **Dynamic Programming (Weighted Job Scheduling)** – maximizes total priority weight
